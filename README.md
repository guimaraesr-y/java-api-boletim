# Project: BoletimAPI
Essa é uma RESTful API fictícia de um sistema de boletim, bem como o de uma escola. O sistema é bem simples e limitado ainda. Este é o meu primeiro projeto de API em Java.

## Tecnologias Utilizadas

As tecnologias que utlizei para o desenvolvimento dessa API foram:

- Java (JDK 19)
- Spring Boot
- H2 (SGBD)
- Lombok
